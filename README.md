# Rayssa Fernanda Massoterapia

Projeto completo do site, preparado para GitHub e Vercel. As imagens, o vídeo da hero, a logo e os prints dos depoimentos estão na pasta `public`.

## Publicar pelo GitHub e Vercel

1. Crie um repositório vazio no GitHub.
2. Envie todo o conteúdo desta pasta para a raiz do repositório.
3. Na Vercel, escolha **Add New > Project** e importe o repositório.
4. Confirme o framework **Next.js**.
5. Mantenha os comandos automáticos da Vercel e clique em **Deploy**.

Não envie apenas um arquivo HTML. Este projeto utiliza React/Next.js para preservar a animação da headline, a otimização das imagens e o comportamento responsivo.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.
